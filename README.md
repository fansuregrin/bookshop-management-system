# Bookshop Management System

## How to use
```bash
git clone https://github.com/fansuregrin/bookshop-management-system.git bookshop-management-system
cd bookshop-management-system
mkdir build
cd build
cmake ..
cmake --build .
./bookshop-management-system
```
